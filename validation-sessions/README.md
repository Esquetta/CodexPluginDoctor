# Validation Sessions

This folder stores structured real-world validation session notes.

Use it to capture:

- target list
- commands run
- output summaries
- true positive / false positive decisions
- tuning recommendations

Recommended flow:

1. Duplicate [`TEMPLATE.md`](./TEMPLATE.md)
2. Rename it with the date and session label
3. Fill it during or immediately after the validation pass

Example filename:

```text
2026-04-27-real-world-validation-wave-01.md
```
