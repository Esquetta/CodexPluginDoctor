# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

Examples of behavior that contributes to a productive project:

- using welcoming and inclusive language
- respecting different viewpoints and experience levels
- giving direct, constructive technical feedback
- focusing criticism on code, behavior, or project impact
- accepting responsibility and correcting mistakes

Examples of unacceptable behavior:

- harassment, insults, or personal attacks
- public or private intimidation
- publishing private information without permission
- sustained off-topic disruption
- other conduct that would be inappropriate in a professional setting

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that violate this code of conduct.

Serious or repeated violations may result in temporary or permanent bans from project spaces.
