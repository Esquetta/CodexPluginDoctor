# v0.1.1 Release Notes

## Summary

`v0.1.1` is the first public npm publication of Codex Plugin Doctor.

This patch release follows the GitHub-only `v0.1.0` release and normalizes npm package metadata so the global CLI executable is preserved by the registry publish flow.

## Release Status

- Version: `0.1.1`
- Status: final
- Package publish: public npm
- Distribution: npm registry, GitHub repository, GitHub Releases, local `npm pack`, local `npm link`
- License: MIT

## Changes Since `v0.1.0`

- normalized package `bin` metadata from `./dist/cli.js` to `dist/cli.js`
- updated README quick start instructions for `npm install -g codex-plugin-doctor`
- updated npm release documentation for first registry publication

## Install

```bash
npm install -g codex-plugin-doctor
codex-plugin-doctor check .
```

## Verification

Run before publishing:

```bash
npm test
npm run build
npm run prepare-release
npm publish --dry-run --access public
```

Run after publishing:

```bash
npm view codex-plugin-doctor version
npm install -g codex-plugin-doctor
codex-plugin-doctor --help
```

## Known Limits

- Runtime probing focuses on command-based MCP servers.
- OAuth/browser authentication flows are out of scope.
- Hosted dashboard, docs website, and web report viewer remain out of scope.
