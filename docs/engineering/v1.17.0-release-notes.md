# v1.17.0 Release Notes

`v1.17.0` adds workflow automation around repeated validation, dependency security checks, and local git hooks.

## Added

- `codex-plugin-doctor watch <path>` runs continuous plugin validation with debounce support.
- `codex-plugin-doctor audit deps <path>` reports dependency vulnerability scan results from npm audit.
- `codex-plugin-doctor init-git-hooks [path]` installs pre-commit and pre-push validation hooks.

## Verification

```bash
npm test
npm run release-check -- --allow-published
npm run verify-release-sync
```
