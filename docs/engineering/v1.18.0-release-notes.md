# v1.18.0 Release Notes

`v1.18.0` formalizes the public automation surfaces introduced in `v1.17.0`.

## Added

- `doctor contract --json` now lists `audit deps`, `watch --json`, and `init-git-hooks --json` as public JSON surfaces.
- `init-git-hooks --json` emits a machine-readable setup report with written hook paths and overwritten hook metadata.
- README examples now cover dependency audits, watch mode, and git hook setup with CI-friendly JSON output.

## Verification

```bash
npm test
npm run release-check
npm publish --access public
```
