# v1.20.0 Release Notes

`v1.20.0` adds a manifest file for GitHub Action report artifacts.

## Added

- Every GitHub Action run now writes `codex-plugin-doctor-action-manifest.json` into the configured report directory.
- The manifest records the installed CLI version, final action status, report directory, artifact name, target mode, and enabled report paths.
- The action exposes `action-manifest-path` for follow-up workflow steps that need to discover generated reports without hard-coded paths.
