# v1.16.0 Release Notes

`v1.16.0` adds a compact review bundle verification mode for CI logs.

## Added

- `doctor review-bundle verify --failures-only` renders the verification header and failed-check summary without the full check log.
- JSON verification output remains unchanged; the new flag only affects text rendering.
- Regression coverage confirms the compact output keeps failed checks and omits passing check noise.

## Verification

```bash
npm test
npm run release-check
npm publish --access public
```
