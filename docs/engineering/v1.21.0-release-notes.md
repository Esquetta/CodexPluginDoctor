# v1.21.0 Release Notes

`v1.21.0` hardens npm child-process execution in release and package audit paths.

## Fixed

- `verify-release-sync` and `release-check` no longer use Node's `shell: true` option when invoking npm commands.
- `doctor npm` and `audit deps` now use the same Windows-compatible npm invocation pattern without shell-based argument concatenation.
- Release verification output no longer emits Node `DEP0190` warnings from these npm command paths.

## Verification

```bash
npm test
npm run release-check
npm publish --access public
```
