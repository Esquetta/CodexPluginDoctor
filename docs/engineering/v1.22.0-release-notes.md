# v1.22.0 Release Notes

`v1.22.0` adds tarball-level evidence to npm preinstall scans.

## Added

- `doctor npm --json` now includes a `tarball` object for the packed archive that was scanned.
- The tarball metadata includes filename, path, integrity, shasum, size, unpacked size, file count, extracted package root, package name, and package version.
- `doctor.npm.json` in the public output contract now lists `tarball` as a required report field.

## Verification

npm test
