# v1.12.1 Release Notes

`v1.12.1` makes release and npm publish validation deterministic after the `v1.12.0` symlink protection release.

## Fix

- Vitest test files now run serially so runtime and performance-sensitive checks do not interfere with each other during `release-check` or `prepublishOnly`.
- The `v1.12.0` symlink and junction escape protection is unchanged.

## Verification

```bash
npm test
npm run release-check
npm publish --access public
```
