# v1.19.0 Release Notes

`v1.19.0` makes bundled validation evidence easier to collect from GitHub Actions.

## Added

- The GitHub Action now accepts `corpus: "true"` to write `validation-corpus.json` into the configured report artifact directory.
- The GitHub Action now accepts `contract: "true"` to write `output-contract.json` into the configured report artifact directory.
- The action exposes `validation-corpus-path` and `output-contract-path` outputs for follow-up workflow steps.

## Verification

```bash
npm test
npm run release-check
npm publish --access public
```
