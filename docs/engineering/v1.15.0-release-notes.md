# v1.15.0 Release Notes

`v1.15.0` makes review bundle verification failures easier to consume in CI and terminal output.

## Added

- Review bundle verification JSON now includes `failedChecks`, a deterministic list of failed check IDs and messages.
- Text verification output now shows a compact failed-check summary before the full check log.
- Regression tests cover empty success summaries, JSON failure summaries, and text failure summaries.

## Verification

```bash
npm test
npm run release-check
npm publish --access public
```
